============
Installation
============


Stable release
--------------

To install Web Content for Humans, run this command in your terminal:

.. code-block:: console

    $ pip install web_content_for_humans

This is the preferred method to install Web Content for Humans, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Web Content for Humans can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ruslan-rv-ua/web_content_for_humans

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/ruslan-rv-ua/web_content_for_humans/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ruslan-rv-ua/web_content_for_humans
.. _tarball: https://github.com/ruslan-rv-ua/web_content_for_humans/tarball/master
